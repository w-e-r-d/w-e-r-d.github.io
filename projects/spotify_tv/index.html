a wee spotify tv

this is a gift i made that connects two spotify accounts' activity.

apologies for the terrible picture. i failed to take a good one before i gave it away.

what does it do?

  • it shows when each account is using spotify. (the green leds turn on)
  • it shows album art for the currently playing track (for both accounts). the black switch in the middle changes between the two accounts.
  • it adds the currently playing track from one account to the other account's queue (this is what the buttons on the left do. one for each direction)
  • that's it!

what's it built with?

  • raspberry pi 4B is the computer. probably overkill for this but it's the only one i wasn't using for anything else.
  • python is the software. i used pillow to do all the album art processing, pygame to render the display and spotipy for the spotify integration
  • lcd display. idk the exact model but i had it laying around from an old project.
  • mdf, chalk paint, and vinyl wrap for the case
  • little things like resistors, buttons, switches, wires, etc.
