spotify keypad (spotipad)

why did i make this?

i made this because when i'm at work i'm listening to spotify. and i often am listening to new music, which i like to save into various playlists if i like the song.

buuut it's a bit disruptive and distracting to leave the window i'm working in, move over to my other monitor, find the playlist i want to add the song to, and drag the song into it.

so i made this keypad to store playlist ids. that way without even lifting focus from my working window i can save whatever song i'm listening to into a number of presaved playlists via the keypad

how does it work?

keypad is built with raspberry pi pico and pimeroni led keypad kit, pico firmware is micropython, the listener software is python built into an exe, and the case is 3d printed

i'll make a tutorial at some point, but i think it might be better as a video rather than a bunch of instructions like my spotify field scanner tutorial. so it might take me a while to figure out the best way to do that

the firmware (just a .py file on the raspberry pi pico) sends instructions to a listener that lives in your system tray

the listener communicates with spotify api or your computer depending what button you pressed (add to playlist vs skip/play/pause)
