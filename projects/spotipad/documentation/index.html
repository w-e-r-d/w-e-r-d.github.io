spotify keypad (spotipad) documentation

heyo. you're probably here because you need help.

so here's a few things that might help. if not, email me!

1: spotify api setup

go to developer.spotify.com and sign in with your spotify account. then go to your dashboard.

in dashboard, click create app

name it whatever you want, describe it however you want. leave website blank. put this exactly into redirect uri: http://127.0.0.1:8000/callback then press add.

check web API box

check that you understand all the stuff box

press save


then just copy the client id and client secret into the spotipad tray app's api settings window

press test connection and it should let you know that you've been signed in. then save and you should be good to go!

3: something else is not working

exit and restart the tray app. unplug and re-plug the usb. if still broken email me.
