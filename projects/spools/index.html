spools.fm

spools is a collection of tools for spotify. it's currently in beta so access is restricted. if you consider yourself a spotify power user and want to try it, email me to get whitelisted: beta@spools.fm

here are some things it can do:

  • sort
  • filter
  • import csv to playlist
  • export playlist to csv
  • build discographies
  • export/import profile playlists
  • deduplicate
  • export album art
  • playlist algebra (union, intersect, subtract)
  • and more!

here are some features i'm working on:

  • workflows: combine a sequence of tools into a 1-click preset or automations
  • canvas: an interactive, visual, and intuative playlist builder/library organizer inspired by miro
  • a playlist cover art generator/builder
