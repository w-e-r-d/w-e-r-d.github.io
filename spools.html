currently testing a desktop version with a limited group of users (that's why there's a client id creation guide below). as soon as i have a build that's somewhat stable, i'll post it to download or (hopefully) as a web app.

How to get a client ID to test Spools

Step 1: go to spotify developer

just search on google. should be the first link

Step 2: sign in

sign in w/ spotify account or create an account

Step 3: create an app

on ur dashboard, click the create app button

Step 4: setup app

make sure redirect is: http://127.0.0.1:8000/callback and u click add

Step 5: copy client id into spools

copy ur client id. go to spools' settings page and put it in the client id field
